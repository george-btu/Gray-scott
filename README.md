# Gray-scott
Material for the paper: Modelling Dynamic Structures of Biological Systems with QPNC
- You can reproduce our results by openning our models (in the folder "models") in Snoopy, and then you choose to run simulation. 
- To reproduce the  video clip, you need to use the tools and scripts in the script folder. First run simulation in Snoopy, export simulation traces to csv file and then use  Snoopy2Gnuplot.jar on csv file as input and showCSV.py to produce video clip.
